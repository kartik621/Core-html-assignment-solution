HTML Tags

  1. <html>: The root element of an HTML page. All other elements are contained within it.
  2. <head>: Contains meta-information about the HTML document, such as title, scripts, styles, and more.
  3. <body>: Represents the content of an HTML document, including text, images, links, and more.
  4. <h1> to <h6>: Heading elements used to define different levels of headings, with <h1> being the highest level and <h6> being the lowest.
  5. <p>: Defines a paragraph of text.
  6. <a>: Creates a hyperlink to another webpage or a specific location within the same page.
  7. <img>: Inserts an image into the HTML document.
  8. <ul>: Represents an unordered list, used to group related items without any particular order.
  9. <ol>: Represents an ordered list, used to group related items with a specific numerical or alphabetical order.
  10. <li>: Defines a list item within an unordered or ordered list.
  11. <table>: Creates a table to organize data into rows and columns.
  12. <tr>: Defines a table row.
  13. <td>: Represents a table cell within a row, containing data or other HTML elements.
  14. <div>: A generic container element used to group and style other HTML elements.
  15. <span>: An inline container element used for styling or grouping inline elements.
  16. <form>: Contains form elements such as input fields, buttons, checkboxes, and more for user input.
  17. <input>: Creates an input field for user data entry.
  18. <button>: Displays a clickable button on the webpage.
  19. <script>: Embeds or references external JavaScript code within an HTML document.
  20. <style>: Embeds or references CSS styles within an HTML document.
